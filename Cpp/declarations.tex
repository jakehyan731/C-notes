\index{C++11}
\subsection{C++11: auto}

\index{C++!STL}
\IFRU{Пользуясь}{When using} \ac{STL}, 
\IFRU{иногда надоедает каждый раз определять тип}{sometimes it is very boring to declare the type of} 
\glslink{iterator}{\IFRU{итератора}{iterator}} \IFRU{вроде}{like}:

\begin{lstlisting}
for (std::list<int>::iterator it=list.begin(); it!=list.end(); it++)
\end{lstlisting}

\IFRU{Тип}{The} \IT{it} \IFRU{вполне можно получить из}{can be deduced from} \TT{list.begin()}, 
\IFRU{поэтому, в начиная со стандарта}{so that is why starting from the} 
C++11
\IFRU{можно использовать}{standard, } \IT{auto}\IFRU{}{ can be used instead}:

\begin{lstlisting}
for (auto it=list.begin(); it!=list.end(); it++)
\end{lstlisting}
