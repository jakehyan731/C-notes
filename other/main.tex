\IFRU{Что хранится в объектных и бинарных}{What is stored in the binary} (.o, .obj, .exe, .dll, .so) \IFRU{файлах}{files}?

{Usually it is only data (global variables) and function bodies (including class methods)}.

\IFRU{Информации о типах (классы, структуры, typedef(\ref{typedef})-ы) там нет}
{There are no type information (classes, structures, typedefs(\ref{typedef})) there}.

\IFRU{В этом заключается одна из больших проблем декомпиляции ~--- отсутствие информации о типах}
{That is one of the serious decompilation problems ~--- type information absence}.
