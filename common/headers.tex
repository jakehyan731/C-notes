\section{\IFRU{Заголовочные файлы}{Header files}}

\IFRU{Заголовочные файлы это описания некоего интерфейса, своего рода документация}
{Header files are an interface description, a documentation in some sense}.
\IFRU{Очень удобно в многооконном редакторе работать со своим кодом в одном окне, а в другом
держать заголовочные файлы для справки}{It is very convenient to work with the code in one editor window
while having header files in another window for reference}.
\IFRU{Поэтому очень полезно делать их похожими на справочник}{It can be said it is advisable
to make header files looks like references}.
